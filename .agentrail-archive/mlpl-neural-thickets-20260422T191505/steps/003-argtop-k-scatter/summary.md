Added argtop_k(values, k) and scatter(buffer, index, value) builtins. argtop_k: k indices of the largest entries in a rank-1 vector, sorted by descending value with tie-break to lower index (stable sort); rejects k>len, k<0, non-rank-1, non-scalar k, wrong arity. scatter: functional rank-1 scalar write returning a new buffer with buffer[index] = value; rejects out-of-range/negative/non-scalar index, non-rank-1 buffer, wrong arity. Both implemented in a new crates/mlpl-runtime/src/ensemble_builtins.rs module dispatched through the existing call_builtin chain -- no eval.rs changes needed. 19 tests (9 argtop_k + 10 scatter).
