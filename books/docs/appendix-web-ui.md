# Appendix: The Web UI

For those who prefer a browser-based environment, MLPL provides a rich Web UI that serves as a modern alternative to Jupyter Notebooks.

## Features

1.  **Instant Execution:** Type code and see results in real-time.
2.  **Visual Trace:** Every step of your computation is visualized as an SVG.
3.  **Persistence:** Your work is saved locally in your browser, allowing you to pick up where you left over.

## Deployment

The Web UI is deployed via GitHub Pages and is available at:
[https://sw-ml-study.github.io/sw-mlpl/](https://sw-ml-study.github.io/sw-mlpl/)

It is compiled from Rust to WebAssembly (Wasm), ensuring that your computations happen locally in your browser with near-native performance.
