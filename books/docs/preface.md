# Preface

This book is born from a desire to bridge the gap between high-level machine learning research and the low-level mechanical sympathy required to execute it efficiently. 

## The Purpose of This Book

The goal of this guide is not just to teach you the syntax of a new language, but to introduce a new way of thinking about data. Machine learning is often obscured by layers of boilerplate and complex frameworks. MLPL aims to strip that away, leaving only the transformations—the "shape-shifting" of data—that defines the field.

Whether you are a researcher looking for a more concise way to express your ideas, a Rust engineer interested in machine learning, or a student of the APL lineage, this book provides the roadmap.

## How to Use This Book

- **The Language Reference** is for when you need to know exactly how a primitive behaves.
- **The User Guide** is for when you want to build something.
- **The Comprehensive Guide** is for those who want to understand the "why" behind the "how."

We encourage you to keep a REPL or the Web UI open as you read. MLPL is a language of action; the best way to learn it is to see the shapes change in real-time.
