Compile-to-rust step 002 shipped. New crate crates/mlpl-lower-rs/ walks mlpl_parser::Expr AST and emits a proc_macro2::TokenStream of Rust. Covers the scalar-arithmetic subset (IntLit, FloatLit, UnaryNeg, BinOp +-*/); emits native f64 arithmetic wrapped at the outermost level in mlpl_rt::DenseArray::from_scalar. Two test suites: tests/lower_scalar_tests.rs (9 fast string-match tests, run every cargo test) and tests/compile_tests.rs (1 end-to-end test, gated by MLPL_LOWER_RS_COMPILE_TESTS=1, writes temp cargo project + shells to cargo build + runs binary + asserts numeric output). End-to-end verified: '1 + 2 * 3' lowers + compiles + runs + prints 7. README documents the test approach. Unsupported constructs return LowerError::Unsupported for step 003+ to fill in.
