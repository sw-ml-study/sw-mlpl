Built interactive 10-lesson tutorial in the browser REPL. Lessons range from scalar arithmetic through to a full logistic regression training run. Each lesson has title, intro prose, clickable example buttons (loads and runs the line in the persistent WasmSession), a 'Try it' prompt, and Prev/Next navigation. Tutorial mode is toggled via a Header button; entering tutorial hides the Welcome banner. Data-driven design (LESSONS const slice) makes it easy to add more lessons. Module layout: tutorial.rs (data + callbacks), components.rs (TutorialPanel), main.rs (state wiring).
