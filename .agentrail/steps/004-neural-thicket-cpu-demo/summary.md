Authored demos/neural_thicket.mlpl: trains a Tiny LM base, sweeps 4 perturbation families x 4 seeds = 16 variants, scores each on a held-out string, builds losses via chained scatter inside per-family for-loops over a length-4 index list (since repeat has no counter), reshapes to a 4x4 heatmap, picks top-K via argtop_k, and averages logits across all 16 for an ensemble. CPU-only; ~10-15s on M-class. Manual run produces ens_metric ~4.6. Integration test in neural_thicket_tests.rs runs a cut-down (V=256, d=8, ctx=4, 5 train steps) end-to-end in <1s.
